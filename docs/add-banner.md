# A universal banner (for repositories, profiles, organizations, StackOverflow)

[![I love watermelon!](https://raw.githubusercontent.com/y0usaf/ILoveWatermelon/main/banner-no-action.svg)](https://github.com/TheBSD/StandWithPalestine/blob/main/docs/README.md)

All banners and badges link to [StandWithPalestine](https://github.com/TheBSD/StandWithPalestine/blob/main/docs/README.md).

1. Add a **`README.md`** file to your repo, StackOverflow or GitHub [profile](https://docs.github.com/en/account-and-profile/setting-up-and-managing-your-profile/customizing-your-profile-readme) or [organization](https://docs.github.com/en/organizations/collaborating-with-groups-in-organizations/customizing-your-organizations-profile).

2. Add a single line on top of it:

```md
[![I love watermelon!](https://raw.githubusercontent.com/y0usaf/ILoveWatermelon/main/banner-no-action.svg)](https://github.com/TheBSD/StandWithPalestine/blob/main/docs/README.md)
```

## Variants

- `banner-no-action.svg` — text banner
- `banner-no-text.svg` — art-only banner
- `banner-direct.svg` — text banner with arrow
